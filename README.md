# DW KI App

Self-hosted KI-Dashboard fuer Chat, Research, Workflows und Agenten.

## Lokaler Start

```bash
npm start
```

Danach im Browser oeffnen:

```text
http://127.0.0.1:4173/
```

## Inhalt

- `index.html` - Dashboard-Oberflaeche
- `style.css` - Layout und Styling
- `main.js` - Interaktion und Demo-Logik
- `serve.js` - kleiner lokaler Static Server

## Status

Das Projekt ist eine rekonstruierte erste Version des frueheren `DW KI Dashboard`.
